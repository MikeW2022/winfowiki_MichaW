# Schleifen und Anweisungen

## if-Anweisung

```java
if(1 > 0) {
    do something;
} else if (i < 0) {
    do something;
} else {
    do something;
}
```

## for-Loop
```java
for (int i=0; i<j; i++) {
    do something;
}
```

## for-each Loop
```java
int[] zahlen = {1, 5, 12, -2};
    
for (int zahl : zahlen) {
    do something;
}
```


## while-Loop
```java
int i = 0;
int j = 5;

while(i < j) {
    do something;
    i++;
}
```
